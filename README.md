# TKinter Contacts Manager App

The goal of this project is to provide minimalistic TKinter project that everyone can use, which just works out of the box and has the basic setup you can expand on.

### Features

* OOP model
* Test datas import  
* Create new Contact
* Modify Contact
* Delete Contact
* Used with SQLite

## Run the project

* from terminal go to the directory in wich you have downloaded the project
* go inside the ContactManager-master folder --> `cd ContactManager-master`
* to create the database with the test datas run the application with the command `python db_create.py`
* run the main view of the application with the command `python app.py`
